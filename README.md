# 🛡️ MindShield - Digital Wellness App

**Combat phone addiction through financial accountability**

## 🎯 Project Overview

MindShield is a cross-platform mobile application that helps users reduce smartphone addiction by implementing financial stakes for focus sessions. Users commit money (₹200-₹2000) to maintain focus, with funds going to charity if they fail.

## ⚡ Key Features

- **Financial Stakes**: Put money on the line to stay focused
- **Real-time Monitoring**: Track phone usage during focus sessions  
- **Charity Integration**: Failed stakes go to RKC_BHARAT initiatives
- **Progress Analytics**: Detailed reports on focus habits
- **Social Challenges**: Compete with friends and community

## 🏗️ Technical Stack

- **Frontend**: Flutter (Cross-platform)
- **Backend**: Firebase + Node.js
- **Database**: Cloud Firestore
- **Payments**: Razorpay (Indian market)
- **Authentication**: Firebase Auth
- **Analytics**: Firebase Analytics

## 💰 Business Model

- 28% commission on stakes
- 52% donated to charity (RKC_BHARAT)
- 20% refunded to users on success
